Next steps

您现在已经熟悉如何创建 stack navigator，在页面上进行配置，在路由之间跳转以及显示全屏模式。 Stack navigator 及其相关 API 将成为你的 React Navigation 工具栏中最常用的工具，但存在一些问题，是它们无法解决。 例如，无法使用 stack navigator 构建基于选项卡的导航 — 为此，你需要使用 TabNavigator 。

The rest of the documentation is organized around specific use cases, so you can jump between the sections under "Guides" as the need arises (but it also wouldn't hurt you to familiarize yourself with them pre-emptively!).

虽然大多数用户不需要这样做，但如果你好奇并希望详细了解 React Navigation 的工作原理，建议你按照“构建自己的导航器”这一章节进行操作。

祝好运！
