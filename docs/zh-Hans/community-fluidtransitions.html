过度动画

Fluid Transitions是一个库，它在使用 react-navigation 的页面之间导航期间提供共享元素转换。

共享元素转换是在导航转换过程中，将一个屏幕中的元素转换为另一个屏幕中的相应元素的可视化。

该库实现了一个称为FluidNavigator的自定义导航器，它实现了所有这些功能和更多其它的可能。

安装

在你的项目中安装 FluidTransitions

npm install react-navigation-fluid-transitions
yarn add react-navigation-fluid-transitions

链接

https://github.com/fram-x/FluidTransitions
