NavigationEvents reference

NavigationEvents 是一个 React 组件, 提供声明式 API 来订阅导航事件。在加载完成（didMount）时订阅导航事件, 并在卸载（unMount）时取消订阅。

Component props

  • navigation - navigation props (optional, defaults to reading from React context)
  • onWillFocus - event listener
  • onDidFocus - event listener
  • onWillBlur - event listener
  • onDidBlur - event listener

事件监听器和 navigation.addListener(...) API同时使用.

例子


const MyScreen = () => ( <view> <navigationevents onwillfocus={payload => console.log('will focus',payload)} ondidfocus={payload => console.log('did focus',payload)} onwillblur={payload => console.log('will blur',payload)} ondidblur={payload => console.log('did blur',payload)} /> {/* Your view code */} </view> );

export default MyScreen; ```
