Next steps

Agora você está familiarizado com a criação de um navegador stack, a configuração nos componentes da tela, a navegação entre as rotas e a exibição de modais de tela inteira. O Stack Navigator e suas APIs relacionadas serão as ferramentas usadas com mais freqüência no seu cinto de ferramentas React Navigation, mas há problemas que elas não resolvem. Por exemplo, você não pode criar navegação baseada em guias usando um navegador stack— Para isso, você precisa usar um TabNavigator.

O restante da documentação é organizado em torno de casos de uso específicos, portanto você pode alternar entre as seções em "Como faço para ...?" como a necessidade surge (mas também não faria mal a você se familiarizar com eles preventivamente!).

Embora a maioria dos usuários não precise fazer isso, se você estiver curioso e quiser saber mais sobre como funciona o React Navigation, recomendamos trabalhar na seção "Criar seu próprio navegador".

Por fim, convém marcar a página Referência da API para referência futura.

Boa sorte!
